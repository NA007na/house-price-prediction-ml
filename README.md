# House Price Prediction using Machine Learning

## Overview
This project predicts house prices based on area using a simple Machine Learning model.

## Technologies Used
- Python
- Google Colab
- scikit-learn
- NumPy

## How It Works
1. Train a Linear Regression model on sample data
2. Take user input for house area
3. Predict house price dynamically

## Learning Outcome
- Understood ML workflow
- Learned model training and prediction
- Gained hands-on experience with Google Colab and GitHub
